# 🔍 ResearchFocus

**ResearchFocus** is a Chrome extension designed for journalists, bloggers, and critical thinkers. It removes the "noise" from the internet, allowing you to focus on the facts.

## ✨ Features
* 🚫 **Clutter Removal:** Automatically hides sidebars, trending lists, and clickbait on major news sites.
* ⚡ **Real-Time Cleaning:** Uses `MutationObserver` to ensure new distractions don't pop up as you scroll.
* 📝 **Instant Blockquotes:** Select any text and press `Alt + C` to generate a formatted WordPress-ready blockquote with the source URL.
* 🌗 **Toggle Mode:** Easily turn the focus mode on or off via the extension popup.

## 🚀 Installation
1. Download or clone this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer Mode**.
4. Click **Load Unpacked** and select this folder.
